HTML Introduction

What is HTML?

  • HTML stands for Hyper Text Markup Language

  • HTML is the standard markup language for creating Web pages

  • HTML describes the structure of a Web page

  • HTML consists of a series of elements

  • HTML elements tell the browser how to display the content

  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

What is an HTML Element?

An HTML element is defined bya start tag, some content, and an end tag

HTML Links - Hyperlinks

HTML links are hyperlinks. You can click on a link and jump to another document.

When you move the mouse over a link, the mouse arrow will turn into a little hand.

A link does not have to be text. It can be an image or any other HTML element.

HTML Links - Syntax

The HTML "a" tag defines a hyperlink. It has the following syntax:

< a href="url">link text< /a>

By default, links will appear as follows in all browsers:

  • An unvisited link is underlined and blue

  • A visited link is underlined and purple

  • An active link is underlined and red

  • when mousing over a link (a:hover)

HTML Links - The target Attribute

The target attribute specifies where to open the linked document.

The target attribute can have one of the following values:

  • _self - Default. Opens the document in the same window/tab as it was clicked

  • _blank - Opens the document in a new window or tab

  • _parent - Opens the document in the parent frame

  • _top - Opens the document in the full body of the window

HTML Images Syntax

The HTML"img" tag is used to embed an image in a Web page.

Images are not technically inserted into a Web page; images are linked to Web pages.

The "img" tag creates a holding space for the referenced image.

The "src" attribute is required, and it specifies the path to the image you want to display.

The "alt" attribute holds a text description of the image, which isn't mandatory but is incredibly useful for accessibility.

The "width" and "height" attributes are optional, and they specify the width and height of the image.

Use the "style" attribute to apply CSS styles to images.

Define an HTML Table

A table in HTML consists of table cells inside rows and columns.

  • Each table row is defined with a < tr> tag. Each table header is defined with a < th> tag. Each table data/cell is defined with a < td> tag.
  • By default, the text in < th> elements are bold and centered.
  • By default, the text in < td> elements are regular and left-aligned.

HTML Table Padding & Spacing

HTML tables can adjust the padding inside the cells, and also the space between the cells.

To add padding on table cells, use the CSS padding property:

th, td { padding: 15px; }

To add padding only above the content, use the padding-top property.

Add padding to the other sides with the padding-bottom, padding-left, and padding-right properties:

th, td { padding-top: 15px; padding-bottom: 15px; padding-right: 50px; padding-left: 50px; }

To add space between the cells, use the border-spacing property:

table { border-spacing: 15px; }

HTML - The Head Element

The HTML <head> element is a container for the following elements: <title>, <style>, <meta>, <link>, <script>, and <base>.

  • The element is a container for metadata (data about data)
  • The <head> element is placed between the <html> tag and the <body> tag
  • The <title> element is required and it defines the title of the document
  • The <style> element is used to define style information for a single document
  • The <link> tag is most often used to link to external style sheets
  • The <meta> element is typically used to specify the character set, page description, keywords, author of the document, and viewport settings
  • The <script> element is used to define client-side JavaScripts
  • The <base> element specifies the base URL and/or target for all relative URLs in a page

HTML Encoding (Character Sets)

To display an HTML page correctly, a web browser must know which character set to use.

The ASCII Character Set

The ASCII character set is a 7-bit set of codes that allows 128 different characters. These characters include uppercase and lowercase letters, digits, and symbols.

The ANSI Character Set

The ANSI character set is a superset of the ASCII character set and is also known as Windows-1252.

The ISO-8859-1 Character Set

The ISO-8859-1 character set is used for Western European Languages.

The UTF-8 Character Set

UTF-8 is the most common character encoding for the World Wide Web.

HTML Uniform Resource Locators

A Uniform Resource Locator (URL) is used to address a document (or other data) on the web.

A web address like https://www.w3schools.com/html/default.asp follows these syntax rules:

  • scheme: defines the type of Internet service (most common is http or https)
  • prefix: defines a domain prefix (default for http is www)
  • domain: The name of the domain (for example w3schools.com)
  • port: The port number to use to access the resource (default is 80)
  • path: The location of the resource on the server (like /html/default.asp)
  • filename - defines the name of a document or resource

Playing a YouTube Video in HTML

YouTube offers an iframe to embed videos. The iframe element is used to insert content from another source, such as an advertisement, into a web page.

Add mute=1 after autoplay=1 to let your video start playing automatically (but muted).

Add playlist=videoID and loop=1 to let your video loop forever.
loop=0 (default) - The video will play only once.
loop=1 - The video will loop (forever).

Add controls=0 to NOT display controls in the video player.
controls=0 - Player controls does not display.
controls=1 (default) - Player controls is displayed.

HTML Plug-ins

Plug-ins are computer programs that extend the standard functionality of the browser.

Plug-ins were designed to be used for many different purposes:

  • To run Java applets
  • To run Microsoft ActiveX controls
  • To display Flash movies
  • To display maps
  • To scan for viruses
  • To verify a bank id
  • And more...

The <object> Element

The <object> element is used to embed different kinds of media, such as graphics, audio, video, Java applets, ActiveX, PDF, and Flash, into an HTML document.

The <embed> Element

Web browsers have supported the element for a long time. However, it has not been a part of the HTML specification before HTML5.
